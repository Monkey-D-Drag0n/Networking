# Basics of Computer Networking

Networking is the Process of Connecting Computer and other devices so they can share data, resources and services.
These Connections allow devices to communicate with each other over short or long distence using physical wires or cable or wirelessly using radio waves or infrared signals.


## Basic Terminologies


### Network :- 
A Network is a collection of computers and devices that communicate with each other to share data, resources and services.
---

### Node :- 
A Node is any devices in a network that can send, recevies or forward data, such as computer, mobile phone, printers, services, and IoT Devices.
---

### Networking Devices :-
Networking Devices are Hardware components that manage control, and support communication within a network such as routers, switches, hubs, and access points.
